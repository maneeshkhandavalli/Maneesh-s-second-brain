### **Definition**

A ternary operator is a **shortcut for `if...else`** that works in a single line.

**Syntax:**
```JavaScript
	condition ? valueIfTrue : valueIfFalse
```
Example:
```JavaScript
	let age = 20;
let status = age >= 18 ? "Adult" : "Minor";
console.log(status); // Adult

```
